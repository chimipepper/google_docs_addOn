Help Page

  • Name
    The name of the REQUIREMENT **Requires Heading Style format**
    Format > Paragraph Styles > Heading ___
    *Title and Subtitle are not supported*

    Indentation Levels
    Heading Style levels will determine the level of indentation that the requirement will have within the Spirateam requirement list
    -The greater the heading value, the greater the indent value will be for that requirement in relation to the previous requirement
    Example:
    Heading 1

    >Heading 2

    >>Heading 3

  • Description
    -Any text following a Heading Style will become the description of that requirement
    -Bold, italic, underline, and colored formatting is currently unable to be imported to Spirateam

  • Comments Not Currently Supported
